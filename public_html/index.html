Slackfind.NET

It's coming!

A major redesign and re-focusing of slackfind.net is underway.

In the past slackfind.net was a leading resource for finding Slackware Linux software packages. Sadly, it fell out of favour for any number of reasons and eventually went dark. Recently, we have begun bringing slackfind.net back to life. So far, the website infrastructure has been put in place. The old python code is being examined/fixed. There should be some real functionality up in a few weeks.

In the meantime, the people over at slackware.pkgs.org have created a fine aggregator of package resources for recent versions of Slackware Linux.
